项目定位

心动伴侣是一个面向 Android 优先的 AI 虚拟伴侣应用，核心目标不是“回答问题”，而是让用户感受到： 被记住、被关心、关系在发展。

核心功能

💬 角色聊天

多角色人设对话，流式输出，支持图文消息与长期会话。

🧠 记忆系统

自动提取关键记忆 + 用户主动补充，支撑长期关系体验。

📔 角色日记

自动生成日报/周记/月记，帮助观察角色对用户关系的沉淀。

🔐 模式管理

Admin 与探索模式分离：创作可控，用户沉浸不被参数打断。

🔔 生命系统

每日通知唤回、主动互动能力，提升回访与留存。

📞 多模态能力

保留语音/视频通话能力，可扩展视觉理解与语音反馈。

技术栈

  • React Native + Expo
  • TypeScript
  • Zustand + AsyncStorage + SecureStore
  • OpenAI-compatible API（DeepSeek / SiliconFlow / 自定义）
