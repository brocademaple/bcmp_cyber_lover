产品心动点

✨ 情绪价值优先

先听懂你，再回应你。重点不是“答题对错”，而是“当下有没有被接住”。

🧠 关系会沉淀

记忆和日记会把你们的互动沉淀下来，聊天越久越像“老熟人”。

🫧 人设不漂移

每个角色都有独立气质和说话风格，不会聊着聊着变成同一个人。

⏰ 她会主动来找你

每日提醒 + 开场白机制，不必每次都由你先开口。

分段式产品叙事

01

第一次打开：被氛围吸引

不是“配置模型”起手，而是先进入角色氛围与关系想象。

02

持续聊天：被理解

短句节奏和情绪回应让用户感觉“她听懂了我，不只是回了我”。

03

长期使用：被记住

记忆与日记把关系变成时间资产，越用越有“只属于你”的内容。
